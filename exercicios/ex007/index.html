Citações Simples

É possivel apresentar citações no código usando a tag < q > que significa citação. Por exemplo na frase a seguir:

Como se diz fulano: o computador é um burro muito rápido

É possivel nesses casos apenas usar as aspas, mas ao usar a tag < q > você da um sentido de que aquilo é uma citação e não apenas uma forma colocando as aspas.
